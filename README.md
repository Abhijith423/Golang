# Golang
**Go (also called Golang)** is an open-source, statically typed, compiled programming language developed at **Google** by **Robert Griesemer, Rob Pike, and Ken Thompson**. It was officially released in **2009** with the goal of making software development **simple, fast, and efficient**, especially for large-scale systems.
